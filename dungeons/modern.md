Ice Pyramid is a frost-themed dungeon with 10 basic enemies, 3 bosses, themed drops, soul currency, crafted upgrades, and final boss rewards.

| Area | Details |
| --- | --- |
| Theme | Frost, ice, chill, freeze, and dungeon crafting. |
| Monster Count | 10 basic enemies plus 3 bosses. |
| Bosses | Frostfang the Hollow, Glaciara the Frost Witch, Vaelthyr the Eternal Frost King. |
| Currency | Soul Fragment, Lost Soul, Warrior Soul, and Frost Soul. |
| Key Materials | Frostbitten Bone, Glacial Shard, Rime Dust, Permafrost Core, Frozen Heart, Crown Shard. |
| Final Utility Drop | Frozen Vault Key. |
