# Events

## Summer Event

The Summer Event runs on its own island, where an event controller rotates through mini-events on a timer — the boss bar shows what's active. Between the heat, the sharks, and the pirates, the island keeps you busy:

| Event | What Happens |
| --- | --- |
| Heat Wave | The Overheated meter climbs faster while you're on the island. |
| BANANARAMA EXTRAVAGANZA | Bananas rain onto the island. |
| Parrot Jail Break | Lost Summer Parrots appear — return them to the **Parrot Trainer** for Summer Tickets. |
| Shark Attack | Summer Sharks circle the island and drop shark materials. |
| Diamond Crabs | Diamond Crabs surface on the sand with diamonds and rare crab meat. |
| Pirate Attack | Pirates storm the shore; they can drop Summer Tickets. |

Cooling snacks (Ice Cream Cones, Ice Cubes, Soda Pop, Popsicles) keep the Overheated meter down, and **Summer Tickets** are the event currency.

## Dungeon Releases

Every new [dungeon](/dungeons) release is an event in itself — watch in-game announcements for opening dates.
