## Active Holiday Event

**Current event:** Summer Event  
**Current as of:** June 22, 2026  
**Main related page:** [Summer Event Items](/items/summer-event)

The Summer Event is run by an island controller that rotates through active events on a 10-minute timer. Players on the island see the current event on the boss bar and interact with the Overheated system, summer drops, event mobs, and ticket rewards.

### Event Rotation

| Event | What Happens | Useful Links |
| --- | --- | --- |
| Heat Wave | Overheated rises faster while players are on the island. | [Abilities](/abilities) |
| BANANARAMA EXTRAVAGANZA | Bananas and bundles of bananas rain onto the island. | [Summer Event Items](/items/summer-event) |
| Parrot Jail Break | Lost parrots can be brought to the Parrot Trainer for Summer Tickets. | [Monsters](/monsters) |
| Shark Attack | Summer Sharks spawn around the island and drop shark materials. | [Summer Event Items](/items/summer-event) |
| Diamond Crabs | Diamond Crabs surface on the sand and can drop rare crab meat. | [Monsters](/monsters) |
| Pirate Attack | Pirates land along the shore and can drop Summer Tickets. | [Monsters](/monsters) |
